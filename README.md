# 🍕 Food Store TS

Aplicación ecommerce desarrollada con **TypeScript**, **Vite** y manipulación del DOM.

El proyecto simula una tienda online de comida con funcionalidades completas de catálogo y carrito de compras persistente utilizando `localStorage`.

---

# 🚀 Características

✅ Catálogo dinámico de productos  
✅ Búsqueda por nombre  
✅ Filtrado por categorías  
✅ Carrito persistente con localStorage  
✅ Incremento y decremento de cantidades  
✅ Eliminación individual y vaciado del carrito  
✅ Renderizado dinámico con TypeScript  
✅ Diseño responsive  
✅ Navegación entre páginas  
✅ Gestión de usuario y logout  

---

# 🛠️ Tecnologías utilizadas

- TypeScript
- Vite
- HTML5
- CSS
